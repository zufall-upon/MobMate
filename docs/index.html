What is MobMate?

MobMate is a local-only STT + TTS (anonymous voice) tool for voice chat. It converts your spoken voice into text using Whisper.cpp, then plays it back as an anonymous synthesized voice.

MobMate は ローカル環境で完結する音声認識＋匿名発声ツールです。 ゲームなどの VC で、自分の声を直接出さずにコミュニケーションできます。

Why MobMate?

Designed for players who usually stay quiet in voice chat, but still want to send quick reactions like “OK”, “Wait”, or laughter.

聞き専になりがちな人でも、短いリアクションや笑い声で VC に自然に参加できることを目指しています。

Main Features

  • Fully local execution (no cloud, stable ping)
  • Speech recognition via Whisper.cpp
  • Optional TTS with VOICEVOX or Windows TTS
  • Natural laugh detection and replacement
  • Designed for short, fast voice chat reactions
  • ローカル動作（クラウド送信なし）
  • Whisper.cpp による音声認識
  • VOICEVOX / Windows TTS 対応
  • 自然な笑い声の検知・変換
  • 短文・リアクション向け設計
